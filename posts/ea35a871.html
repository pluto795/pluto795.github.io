3.模组使用指南

EasyAuth

CF：https://www.curseforge.com/minecraft/mc-mods/easyauth

服务器登录模组，在玩家首次进入服务器需要注册账号，以后当玩家IP变动时需要重新登录。

注册账号：/register 密码 确认密码

登录账号：/l 密码

InvView

MC百科：https://www.mcmod.cn/class/5193.html

管理员可以打开并操作在线或离线玩家的背包和末影箱，默认OP可使用，支持权限设置。

打开玩家背包：/view inv <玩家名>。

打开玩家末影箱：/view echest <玩家名>。

Ledger

MC百科：https://www.mcmod.cn/class/5389.html

记录玩家破坏、互动、放置方块和杀死生物的记录，类似查询插件，大量硬盘读写需要高IOPS，不建议用HDD硬盘使用该模组。

Hey That’s Mine

MC百科：https://www.mcmod.cn/class/5172.html

保护放置的箱子、木桶、潜影盒、熔炉、高炉、烟熏炉，使只有放置它的玩家可以打开。

  • 容器设置：

/htm set PUBLIC：运行命令后，敲击选择把容器设为公开。

/htm set PRIVATE：运行命令后，敲击选择把容器设为私有。

/htm set KEY：运行命令后，拿着任意物品敲击容器，其他人需使用敲击时的物品作为钥匙才能打开。

以上三条指令的权限节点：htm.command.set（玩家默认拥有）。

  • 允许和取消：

/htm trust 玩家名 global：允许玩家打开你的容器。

/htm untrust 玩家名 global：取消玩家打开你容器的权限。

以上两条指令的权限节点：htm.command.trust（玩家默认拥有）。

  • 移除保护：

/htm remove：移除该容器的保护（htm.command.remove，玩家默认拥有）。

/htm transfer 玩家名：把敲击容器的保护权转移给其他玩家（htm.command.transfer，玩家默认拥有）。

  • 管理权限：

htm.admin：拥有该权限节点后，可无视保护随意打开或破坏容器（OP默认拥有）。

LuckPerms

MC百科：https://www.mcmod.cn/class/5192.html

大名鼎鼎的LuckPerms插件的官方模组版本，使用方法与插件类似。

首先我们需要先在控制台获得最高权限，

使用“lp user <玩家名> permission set luckperms.* true”这个指令可获得最高权限。

如若未取得最高权限，使用指令后只会出现模组的版本号。

给玩家权限：/lp user <玩家> permission set 权限 true

取消玩家权限：/lp user <玩家> permission set 权限 false

创建权限组：/lp creategroup <权限组名>

给组添加权限：/lp group <权限组> permission set 权限 true

取消组权限：/lp group <权限组> permission set 权限 false

继承组：/lp group <权限组> parent add <要继承的组>

将玩家添加到组内：/lp user <玩家> parent add <权限组名>

MCBBS教程：https://www.mcbbs.net/thread-716368-1-1.html

Easywhitelist

CF：https://www.curseforge.com/minecraft/mc-mods/easywhitelist

白名单模组，让离线服务器也可以使用原版的白名单，安装模组后在

“server.properties”文件中的“white-list=false”选项，设为“true”打开服务器白名单。

进入游戏输入“/easywhitelist add|remove 玩家名”即可将玩家添加或移除白名单。

Gunpowder Market

CF：https://www.curseforge.com/minecraft/mc-mods/gunpowder-market

市场模组，用于服务器的玩家交易，需要gunpowder-currency作为前置货币使用。

输入“/market”打开市场页面，进行购买物品，输入“/market add <价格> <数量>”将手里的物品

放到市场中售卖。

Gunpowder Currency

CF：https://www.curseforge.com/minecraft/mc-mods/gunpowder-currency

货币模组，用于服务器转账、交易，输入“/pay <玩家> <金额>”进行转账，

/balance <玩家名>：查看账户余额。

/balance add|remove|set <玩家> <金额> ：给/扣/设置玩家的钱（仅限OP使用）。

/balance top：列出余额前10名的玩家。

/pay <玩家名> <金额>：转账。
